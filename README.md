**COMPUTATIONAL PRACTICES: VISUALISATION & SENSING (IU000116)**

# Nikita Collins - Assessment Submission #

This folder contains all material for submission for the Visualisation & Sensing module assessment.

Exploring how sound data can be visualised in a creative and emotional way.

Dataset used is from NoiseCapture, part of the Noise-Planet project
https://noise-planet.org/noisecapture.html 




File description:

data folder - contains dataset files needed to run project

charts folder - pngs generated during early data exploration

index.html - main file, run this to view website

script.js - code for website execution and interaction

style.css - stylesheet containing all css code for the website

V^0S DOCUMENTATION - documentation for whole design process
